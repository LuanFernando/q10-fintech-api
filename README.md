## Q10 Fintech (API)

### Descrição

-   API desenvolvida usando Laravel
-   Consumindo Gemini AI

### Principais recursos

-   Account (CRUD)
-   Contact (CRUD)
-   Transaction (CRUD)
-   Chat AI

### Para rodar o projeto rode o comando abaixo ou configure o IIS e aponte para o diretório public.

-   php artisan serve
